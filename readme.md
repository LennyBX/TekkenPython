## Init du projet Django Tekken

* Project with django 
* An App Web

### Créer un environement virtuel 
* ```python3 -m venv .venv```
### Activer l'environnement virtuel
* ```source .venv/bin/activate```
### Run this command à la racine
* ```python3 manage.py runserver```
### Go sur cette url
* ```http://127.0.0.1:8000/jeux1/```
